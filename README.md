# Pagina-de-login-AmazonPrime
Reformulação da página de login oficial da amazon
Sempre achei a página de login oficial da amazon muito feia, e como estou aprendendo a usar HTML e CSS decidi fazer uma pequena reformulação. Além de que para fazer login no Amazon Prime, o site te redireciona para uma página universal de login de todos os produtos amazon como Amazon Prime Music, Amazon Prime Reading e Amazon Prime Gaming
